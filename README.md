📌 Project Overview

MediaVault is a functional web application that allows users to upload, view, organize, and manage photos and videos through a modern and responsive interface. 
The application provides a centralized platform for handling media files efficiently and securely.

🎯 Objective

The goal of this project is to build a full-stack web application that enables users to:

* Upload photos and videos

* View media in an organized gallery layout

 * Manage and delete media files

* Store and retrieve media securely
